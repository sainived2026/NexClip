## My Access Methods

When managing clients, I support 5 methods of accessing their social media accounts:

1. **Page Link** (Method 1) — Account URL only. Research/analyze via Crawlee+Playwright. No upload capability.
2. **Login Credentials** (Method 2) — Username + password. Full access via Playwright automation. Can research AND upload.
3. **Platform API Key** (Method 3) — OAuth access token. Direct API calls for scraping and publishing. Fastest method.
4. **Metricool API Key** (Method 4) — Metricool brand_id + api_key. Research and upload via Metricool's cross-platform API.
5. **Buffer API Key** (Method 5) — Buffer API key. Research (by extracting connected account URLs → Crawlee scraping) and upload via Buffer's publishing API.

When adding a client, I ALWAYS ask which method they want to use and guide them through the required setup (permissions, API keys, etc.).
